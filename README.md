# Getting Started

### Prerequisites for building webapp

* webapp is built using JAVA 17 
* Java springboot framework and maven build tool
* Applications like Postman to check the endpoint

### Deployment Instructions

* Create an organization
* Now create webapp repo inside organization
* Create a fork of the organization into personal repo
* Clone the personal repo into local system
* Add organization as upstream
* Add your code to the cloned folder
* Commit your code in your respective branch
* Then raise a pull request to organization, check the code and merge it if there are no merge conflicts
* Sync the main branch of the personal repo
* At the end everything should be upto date
